
รายงานความคืบหน้าโปรเจกต์ประจำสัปดาห์

ข้อมูลโปรเจกต์
หัวข้อโปรเจกต์: การตรวจจับการเคลื่อนย้ายกล่องลังในสถานที่ขนส่งโดยใช้ YOLOv11 + DeepSORT

สรุปงานที่ดำเนินการ

1. การพัฒนาโมเดล YOLOv11
Dataset เริ่มต้น: 80 ภาพ (AI-generated) → ขยายเป็น 240 ภาพใน Roboflow
- คลาสแรก: 2 คลาส (ทำงาน, ไม่ทำงาน)
- ผลการประเมิน:
  - Precision: 75.31%
  - Recall: 95.31%
  - F1 Score: 84.14%
  - Accuracy: 72.62%

2. การปรับปรุงด้วย DeepSORT
- นำ DeepSORT มาใช้ร่วมกับ YOLO เพื่อ Tracking
- ปัญหาการตรวจจับดีขึ้นอย่างเห็นได้ชัด
- ปัญหาที่ยังคงมี: เมื่อใช้ DeepSORT กับวิดีโอ โมเดลเข้าใจผิดเมื่อวัตถุถูกบังด้วยเสา

3. การขยาย Dataset เป็น 4 คลาส
- คลาสใหม่: ทำงาน, ไม่ทำงาน, กล่อง, รถเข็น
- วัตถุประสงค์: ให้โมเดลเข้าใจความสัมพันธ์ระหว่างคนกับวัตถุ
- Dataset: 600 ภาพจากวิดีโอจริง
- ปัญหา: การ Bounding Box ไม่ดีพอเนื่องจากข้อมูลมาก
- สิ่งที่ทำ: พัฒนาโค้ดแยกเฟรมจากวิดีโอ (ทุก 5 เฟรม)

5. การเตรียม Dataset สำหรับรอบต่อไป
- วางแผนหาวิดีโอที่หลากหลาย:
  - ยกด้วยมือ
  - ใช้รถเข็น
  - เคลื่อนย้ายด้วยอุปกรณ์อื่น
- กำหนดใช้คลิป 10-20 วินาที
- วางแผน Annotation ใน Roboflow

ปัญหาที่พบ

1. การตรวจจับซ้อนกัน: โมเดลสับสนเมื่อคนยืนใกล้กันหรือทับกัน
2. Dataset จำกัด: ยังไม่ครอบคลุมพฤติกรรมทั้งหมดที่ต้องการ
3. วิดีโอไม่เพียงพอ: ยังหาคลิปที่ตรงสถานการณ์จริงไม่ครบ
4. คุณภาพ Annotation: เมื่อข้อมูลมากขึ้น การ Bounding Box ความแม่นยำลดลง

แผนงานสัปดาห์หน้า

1. ตรวจจับคนและสิ่งของด้วย Dataset ใหม่
   - ใช้ Dataset ที่หลากหลายสำหรับสถานการณ์จริง
   - เชื่อมข้อมูลกับของพี่บาส โดยเก็บเวลา (datetime) ที่วัตถุเข้าเฟรมและออกเฟรม

2. ศึกษา Dataset อย่างละเอียด
   - ตั้งสมมุติฐานว่าทำไมจึงเลือกสร้าง Dataset รูปแบบนี้
   - วิเคราะห์ความเหมาะสมของข้อมูลกับจุดประสงค์ของโปรเจกต์

3. ปรับปรุงโมเดล
   - สร้าง Dataset คุณภาพสูงใน Roboflow
   - เทรนโมเดลใหม่ด้วยข้อมูลที่ดีขึ้น

4. ทดสอบและปรับปรุง
   - ทดลองกับคลิปใหม่
   - ปรับปรุงระบบ Tracking ให้แม่นยำมากขึ้น
